# Sistema de Inventario para Farmacias

---

## Descripción
Este software de inventario está netamente dirigido a farmacias, es una 
aplicación de escritorio desarrollada en Java con base de datos MySQL para la 
gestión eficiente de productos, stock y reportes.

## Características
- Gestión de productos (agregar, editar, eliminar).
- Registro de Venta de Productos.
